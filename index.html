// CORE CAPABILITIES

Everything a Blue Team Needs

Dibangun dari ground-up untuk defender — bukan port dari tool offensive yang dipaksain jadi defensive.

[1] RECON
Network Reconnaissance
Subdomain enum, port scanning, service fingerprinting, dan asset discovery — semua dalam satu command chain.
[2] VULN
Vulnerability Assessment
Real-time CVE matching, CVSS scoring, dan remediation guidance. Database sync tiap 6 jam dari NVD.
[3] HUNT
Threat Hunting
Query log terstruktur dengan SIGMA rules built-in. Deteksi lateral movement, persistence, dan C2 callbacks.
[4] INTEL
Threat Intelligence
IOC enrichment via VirusTotal, Shodan, AbuseIPDB. Otomatis korelasi dari MITRE ATT&CK framework.
[5] RESP
Incident Response
Playbook otomatis, timeline builder, dan evidence collector. IR workflow dari deteksi sampai remediation.
[6] REPORT
Automated Reporting
Generate laporan executive dan teknikal dalam sekali klik. Template NIST, ISO 27001, dan custom.
